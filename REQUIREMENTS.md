# 保険 vs 投資 比較検証Webアプリ 要件定義

## 概要

ドル建て保険などの貯蓄型保険を「投資商品」として客観的に評価するためのWebアプリ。
保険営業から提示される数字（例：10年後に118%）を分解・検証し、他の資産運用手段と比較できるようにする。

---

## 1. 年率換算計算機能

### 目的
「10年で118%」のような返戻率を、年率（複利）に換算して正しく理解する。

### 実装内容
- **入力項目**
  - 払込方法（一括 / 月払い / 年払い）
  - 払込期間（年）
  - 据置期間（年）（払込終了後〜受取までの期間）
  - 返戻率（%）（例: 118%）
  - 払込保険料総額（任意、シミュレーション用）
- **計算ロジック**
  - 一括払い: `年利 = (返戻率)^(1/運用年数) - 1`
  - 分割払い: IRR（内部収益率）で算出。毎月/毎年のキャッシュフローから正確な年率を計算
- **出力**
  - 年率（複利ベース）
  - 単利換算の参考値
  - 運用年数ごとの資産推移グラフ

### 補足
- 保険の「予定利率」と実質利回りの違いを解説表示
  - 予定利率は保険料全額に適用されるわけではない（手数料・死亡保障コスト控除後の積立部分にのみ適用）
  - 実質利回り = (受取額 - 払込総額) / 払込総額 / 運用年数

---

## 2. 為替リスク分析機能

### 目的
過去の為替データを基に、円換算した場合のリターン変動を可視化する。

### 実装内容
- **為替データ取得**
  - API: Frankfurter API（無料、APIキー不要、ECBデータ）または ExchangeRate-API
  - 取得範囲: 1999年〜現在のUSD/JPYレート（日次）
  - ローカルキャッシュ対応（API負荷軽減）
- **分析・可視化**
  - 過去の任意時点で加入したと仮定した場合の円建てリターンシミュレーション
    - 例: 2014年に加入 → 2024年に受取の場合の為替影響
  - 為替レートの推移グラフ（折れ線）
  - N年間保有した場合の為替変動分布（ヒストグラム）
    - 例: 過去データから「10年保有」のケースを全期間分スライドして集計
  - 損益分岐為替レートの計算と表示
    - 加入時レートに対して、何円まで円高になると元本割れするか
  - 最悪ケース / 最良ケース / 中央値の表示
- **為替シナリオシミュレーション**
  - ユーザーが将来の為替レートを手動入力して円建てリターンを計算
  - 円高5%/10%/20%シナリオの自動計算

## 3. 無リスク資産・他商品との比較機能

### 目的
同じ資金・期間で他の選択肢に投資した場合と比較し、保険の相対的な位置づけを明確化する。

### 比較対象
| 資産クラス | 具体例 | データソース |
|---|---|---|
| 日本国債 | 個人向け国債（変動10年） | 財務省公表利率 |
| 米国債 | 米国10年債利回り | FRED API等 |
| 定期預金 | メガバンク定期預金金利 | 手動入力 or 参考値 |
| インデックス投資 | S&P500 / 全世界株式 | Yahoo Finance等 |
| 新NISA枠活用 | 同額をNISAで運用した場合 | 上記インデックスの税制優遇込み |

### 実装内容
- 同一条件（金額・期間）での最終資産額を並べて棒グラフ表示
- リスク（標準偏差）とリターンの散布図
- 税制の違いを考慮（保険の一時所得控除 vs NISA非課税 vs 通常の20.315%課税）
- 手数料・コストの内訳比較表
  - 保険: 契約手数料、為替手数料（往復）、解約控除、死亡保障コスト
  - 投資信託: 信託報酬、売買手数料

---

## 4. コスト構造の見える化

### 目的
保険商品に内包される各種コストを分解して表示する。

### 実装内容
- **入力項目**
  - 為替手数料（片道、例: 0.25円〜1円）
  - 解約控除率（早期解約時のペナルティ）
  - 予定利率（公表値）
  - 保険関係費用（概算 or 商品名から推定）
- **出力**
  - コストの積み上げグラフ（どの費用がいくら利回りを削っているか）
  - 「見かけの利回り」と「実質利回り」の差を視覚的に表示
  - 損益分岐年数（何年保有すれば元本を回収できるか）

---

## 5. 総合判定レポート

### 目的
入力情報を基に、保険商品の投資としての評価をまとめて表示する。

### 実装内容
- スコアカード形式での表示
  - 実質年利
  - 為替リスク（過去データベースのVaR等）
  - 流動性（解約控除期間の有無）
  - 税制メリット
  - 他商品との相対評価
- PDF/画像エクスポート機能（検討用資料として保存可能）

## 技術スタック（案）

| 項目 | 技術 |
|---|---|
| フロントエンド | Next.js (React) + TypeScript |
| UIライブラリ | Tailwind CSS + shadcn/ui |
| グラフ描画 | Recharts or Chart.js |
| 為替データAPI | Frankfurter API（無料・キー不要） |
| 計算ロジック | クライアントサイドJS（IRR計算等） |
| デプロイ | Vercel（Next.jsとの親和性） |

---

## データソース

| データ | ソース | 備考 |
|---|---|---|
| USD/JPY為替レート | [Frankfurter API](https://www.frankfurter.app/) | ECBデータ、無料、キー不要 |
| 米国債利回り | [FRED](https://fred.stlouisfed.org/) | セントルイス連銀、要APIキー（無料） |
| S&P500過去データ | Yahoo Finance等 | 非公式API or CSV |
| 日本国債利回り | 財務省Webサイト | CSV公開あり |
| 保険商品情報 | ユーザー手動入力 | 各社パンフレットから転記 |

---

## 画面構成（案）

1. **トップページ** - アプリの説明と入力フォーム
2. **年率換算結果** - 返戻率→年率の変換結果
3. **為替リスク分析** - 過去データに基づく為替変動の影響
4. **比較ダッシュボード** - 他商品との横並び比較
5. **総合レポート** - 全分析結果のまとめ

---

## MVP（最小実装）の優先順位

1. 年率換算計算（コア機能、APIなしで実装可能）
2. 為替リスク分析（Frankfurter APIのみで実装可能）
3. 無リスク資産との簡易比較（ユーザー入力ベース）
4. コスト構造の見える化
5. 総合判定レポート・エクスポート

---

## 参考情報

- 金融庁 共通KPI（2025年3月末）: 外貨建て保険5年以上保有の平均 → 年率コスト1.4%、年率リターン7.5%
- 外貨建て保険の予定利率目安: 米ドル建て 2〜3%（円建ては0.25%程度）
- 予定利率 ≠ 実質利回り: 予定利率は保険料から手数料・保障コストを差し引いた積立部分にのみ適用される
